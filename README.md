# 🌌 Aura AI - Assistente Inteligente v2.0

Aura é uma aplicação de chat moderna, rápida e minimalista, desenvolvida com **Next.js 14** e alimentada pelo modelo **Llama 3**. O projeto foca em uma experiência de usuário (UX) fluida, apresentando uma interface profissional inspirada nas melhores ferramentas de IA do mercado.

## ✨ Funcionalidades

* **Interface Premium:** Design limpo e profissional utilizando Tailwind CSS com efeitos de Glassmorphism.
* **Modo Escuro Nativo:** Alternância suave entre temas Light e Dark com persistência via LocalStorage.
* **Gestão de Memória:** Função para limpar histórico de conversas e resetar o estado da aplicação.
* **Totalmente Responsiva:** Adaptada para dispositivos móveis, tablets e desktops.
* **Personalidade Customizável:** Configuração de sistema para definir o comportamento da IA.
* **Alta Performance:** Renderização otimizada e componentes leves para respostas ultrarrápidas.

## 🛠️ Tecnologias Utilizadas

* [Next.js 14](https://nextjs.org/) - Framework React para produção.
* [Tailwind CSS](https://tailwindcss.com/) - Estilização baseada em utilitários.
* [Lucide React](https://lucide.dev/) - Biblioteca de ícones elegantes.
* [Llama 3 (Groq/API)](https://llama.meta.com/) - Modelo de linguagem de última geração.
* [TypeScript](https://www.typescriptlang.org/) - Tipagem estática para maior segurança do código.
